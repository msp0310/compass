# 体制

メンバー別の作業量、週別負荷、案件横断の稼働状況を確認する画面です。

![体制画面](/help/screenshots/resource-view.png)

## 機能概要

- メンバーごとの週別作業量を表示します。
- このプロジェクトまたはチーム横断の負荷を切り替えられます。
- 高負荷のメンバーや空き状況を確認できます。
- タスクの担当割り当てや日程見直しの判断材料に使います。

## 状態の見方

- セル内の時間は、その週に割り当てられている作業量です。
- 色の強さは稼働率の高さを示します。
- 100%を超える場合は負荷超過です。
- 非稼働日や休止中メンバーは新規割り当て候補から外れます。

## 操作方法

- スコープをチーム横断にすると、複数案件の負荷を比較できます。
- メンバー行から担当タスクを確認します。
- 必要に応じてガントへ戻り、日程や担当者を調整します。
