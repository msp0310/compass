# 作業時間

運用保守、問い合わせ、障害対応、開発作業などの実績時間を案件単位で記録します。

## 機能概要

- 日付、担当者、分類、時間、内容、関連タスク・課題を記録できます。
- 内容と補足はMarkdownで記述できます。
- 日報から連携された実績は日報を正本として表示します。
- ログや資料は添付ファイルとして残せます。
- 一般メンバーは自分の作業時間だけを追加・更新・削除できます。PM・PLは案件全体を管理できます。

## 操作方法

- 作業時間追加から入力画面を開きます。
- 実績を関連タスクまたは課題へ紐づけます。
- 添付は許可形式・サイズ・ファイル内容を検証して保存されます。
- 自分の添付は削除でき、PM・PLは案件内の添付を管理できます。
