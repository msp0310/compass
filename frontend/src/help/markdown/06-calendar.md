# カレンダー

案件で利用する稼働日、会社休日、タスク配置を確認する画面です。

## 機能概要

- 案件カレンダーを確認できます。
- 会社休日や非稼働日を考慮したスケジュール管理に使います。
- 国民の祝日は管理設定のカレンダーマスタから取り込めます。

## 状態の見方

- 稼働曜日は作業可能な曜日です。
- 会社休日は作業対象外の日付です。
- タスクが非稼働日にまたがる場合、工数計算では非稼働日を除外します。

## 操作方法

- 日付を見てタスクの集中や休日付近の予定を確認します。
- 休日設定を変更する場合は管理設定のカレンダーへ移動します。
- タスク日程の変更はガントで行います。
