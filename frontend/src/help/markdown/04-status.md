# 概要

選択中案件の進捗、遅延、節目、健全性を確認するダッシュボードです。

![概要画面](/help/screenshots/project-overview.png)

## 機能概要

- 案件全体の進捗と完了率を確認できます。
- 遅延タスク、次のマイルストーン、工程別進捗を確認できます。
- 健全性の確認事項から、問題のあるタスクへ移動できます。

## 状態の見方

- 遅延タスクは、状態が遅延のタスク数です。
- 完了率は完了タスクの割合です。
- 次のマイルストーンは未完了の節目のうち直近のものです。
- 健全性エラーは、担当未設定、非稼働日、依存関係、負荷超過などを示します。

## 操作方法

- 健全性の項目を選択すると、対象タスクへ移動します。
- 進捗や遅延の傾向を確認して、ガントで詳細を編集します。
- 保存前レビューがある場合は、差分を確認してから保存します。
