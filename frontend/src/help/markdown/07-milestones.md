# マイルストーン

案件内のマイルストーン、承認日、リリース判定などを管理する画面です。

## 機能概要

- マイルストーンを一覧で確認できます。
- 新しいマイルストーンを追加できます。
- ガント上のマイルストーンタスクと連動します。

## 状態の見方

- 未着手、進行中、完了、遅延の状態を確認できます。
- 日付が近いものは優先して確認します。
- 次のマイルストーンは概要画面や案件一覧にも表示されます。

## 操作方法

- マイルストーン追加から新しいマイルストーンを作成します。
- 一覧から対象を選択するとガント上の位置を確認できます。
- 完了したマイルストーンは状態を完了に変更します。
