# 案件設定

選択中案件の名称、チーム、期間、メンバー、ステータスを管理する画面です。

![案件設定](/help/screenshots/project-settings.png)

## 機能概要

- 案件名、ライフサイクル状態、期間、次のマイルストーンを設定できます。
- 案件に参加するメンバーを管理できます。
- メンバーごとにPM、PL、メンバー、閲覧者の案件権限を設定できます。
- 完了済み、アーカイブなど案件の扱いを整理できます。

## 状態の見方

- 計画、進行中、完了済みは案件一覧の絞り込みに使われます。
- 案件メンバーは担当候補や体制表示に影響します。
- アーカイブした案件は通常の一覧から外れます。

## 操作方法

- サイドメニューの案件設定から開きます。
- 設定を変更したら保存します。
- チームやメンバー構成の共通マスタは管理設定で編集します。
- 案件設定はPM / オーナーまたはシステム管理者だけが操作できます。
