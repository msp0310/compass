# 管理設定

チーム、メンバー一覧・ログイン、カレンダーを管理する画面です。

![管理設定](/help/screenshots/user-management.png)

## 機能概要

- チームマスタでは、チーム名、記号、所属メンバーを管理します。
- メンバーマスタでは、氏名、ロール、略称、キャパシティ、表示色、非稼働日を管理します。
- メンバー一覧・ログインでは、メンバー基本情報とメールアドレスを一覧で確認し、ログイン可否、権限、パスワード再設定は詳細で管理します。
- カレンダーでは稼働曜日、会社休日、国民の祝日取り込みを管理します。

## 状態の見方

- 有効メンバーは新規担当候補に表示されます。
- 休止中メンバーは既存履歴や既存担当には残ります。
- メンバー権限は管理者、マネージャー、メンバーの3段階です。
- メンバー一覧・ログインのログイン設定とパスワード再設定は管理者だけが操作できます。

## 操作方法

- サイドメニューの管理設定から開きます。
- チーム、メンバー、カレンダーのタブを切り替えます。
- メンバー行でメール、権限、ログイン可否を編集して保存します。
- パスワード欄を空のまま再設定すると仮パスワードが発行されます。
- 祝日取込で国民の祝日をカレンダーに追加します。
