# 権限とセキュリティ

職種 `PM / PL / SE / FE / BE / QA` と操作権限は別に管理します。

## システム権限

- **システム管理者**: 全案件、メンバーアカウント、監査ログを管理できます。
- **一般ユーザー**: チーム・案件で付与された権限に従います。

## チーム権限

- **チーム管理者**: 自チームの所属と要員計画を管理できます。案件ロールがなければガント計画は変更できません。
- **チームメンバー**: 所属チームの許可された案件を利用します。

## プロジェクト権限

- **PM / オーナー**: 計画、案件設定、基準計画、要員を管理できます。
- **PL / 計画編集**: タスク階層、日程、担当を編集できます。案件アーカイブや基準計画承認はできません。
- **メンバー**: 自分の担当タスクの状態、進捗、実績開始日・終了日を入力できます。課題、コメント、自分の作業時間も扱えますが、計画日は変更できません。
- **閲覧者**: 案件を参照できますが更新できません。

## 認証

- セッションはHttpOnly Cookieで管理し、変更APIはCSRFトークンを検証します。
- 初回ログインとパスワード再設定後は、12文字以上の新しいパスワードへ変更します。
- ログイン失敗が続くと一定時間試行が制限されます。
