# 履歴

タスク変更、保存前レビュー、スケジュール変更の兆候を確認する画面です。

## 機能概要

- タスク変更や設定変更の差分を確認できます。
- 保存前レビューでは追加、更新、削除を確認できます。
- 将来的なPM分析として、スケジュール変更回数や変更傾向を蓄積する想定です。

## 状態の見方

- 変更レビューは現在状態と保存済み状態の差分です。
- 変更項目には開始日、終了日、担当、状態、進捗などが含まれます。
- スケジュール変更が多いタスクは、見積もりやリスク管理の材料になります。

## 操作方法

- 差分行を確認し、必要なら対象タスクへ移動します。
- 保存前に変更内容をレビューします。
- 問題なければ保存を実行します。
