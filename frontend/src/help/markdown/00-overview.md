# ヘルプの使い方

このヘルプは、SI案件のスケジュール管理を行うための画面別ガイドです。左メニュー、上部のチーム・案件選択、案件内タブ、管理設定の関係を確認しながら利用してください。

![プロジェクト管理アプリの全体像](/help/screenshots/project-gantt.png)

## 機能概要

- チームごとに案件を管理します。
- 案件ごとにガント、概要、体制、カレンダー、マイルストーン、履歴を確認できます。
- 管理設定では、チーム、メンバー一覧・ログイン、カレンダーを管理します。
- ログインしていない状態ではアプリ本体は表示されません。

## 状態の見方

- 上部の保存状態は、現在の変更が保存済みか未保存かを示します。
- 左メニューの選択状態は、今見ている画面の位置を示します。
- 案件内のタブは、選択中の案件に対するビューです。
- 管理設定はワークスペース全体に対する設定です。

## 操作方法

- 左メニューから案件一覧、案件内ビュー、管理設定、ヘルプを移動します。
- 上部のチーム・案件選択から、作業対象を切り替えます。
- 保存ボタンで現在の変更を保存します。
- 右上のアカウントからログアウトできます。
