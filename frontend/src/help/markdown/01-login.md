# ログイン

メールアドレスとパスワードでログインする画面です。ログインしていない場合、案件一覧やガントなどの画面は表示されません。

![ログイン画面](/help/screenshots/login.png)

## 機能概要

- ログインが有効なメンバーだけがアプリを利用できます。
- メンバーには管理者、マネージャー、メンバーの権限があります。
- 管理者はメンバーのログイン設定とパスワード再設定を行えます。

## 状態の見方

- 入力エラーや認証エラーはログインフォーム内に表示されます。
- セッションが切れた場合はログイン画面に戻ります。
- 初期アカウントはデモ用です。本運用では管理者がメンバーごとにログインを発行します。

## 操作方法

- メールアドレスを入力します。
- パスワードを入力します。
- ログインを押します。
- ログアウトはアプリ右上のアカウントメニューから実行します。
